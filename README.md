# EVtoSP Calculator

## 概要

EVtoSP Calculator は，ポケットモンスターシリーズにおける努力値（EV）を，Pokémon Champions 側のステータスポイント（SP）に換算するためのコンソールアプリケーションです．

目標とするSP配分を入力すると，そのSPを満たすために必要なEV配分を計算し，候補として表示します．

本ツールは，当初，Pokémon Champions 側でSPを直接獲得するコストが高いことを想定し，Pokémon SV 側で育成した個体を連れてくることで育成コストを抑える目的で作成しました．  
現在は，Pokémon Champions 側のSP獲得コストが下がったため，実用上の必要性は低下しています．

そのため，本プロジェクトは主に以下を目的とした学習用ミニプロジェクトとして位置付けています．

- Javaによるコンソールアプリケーション開発
- 入力値チェックの実装
- 目標値から必要値を逆算する処理の設計
- 複数候補を探索する最適化処理の練習
- Git / GitHub を用いたソースコード管理

## 使用技術

- Java
- IntelliJ IDEA Community Edition
- Git
- GitHub

## 主な機能

- 目標SPの入力
- 入力されたSPの妥当性チェック
- SPから必要EVへの逆算
- EV配分候補の生成
- 合計EVの計算
- コンソール上への結果表示
